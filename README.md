# Telegram Bot для записи на занятия

Бот позволяет клиентам записываться на занятия, а администратору управлять слотами и просматривать записи.

## Основные функции

### Для клиентов:
- Регистрация (ввод фамилии и имени, данных ребенка, телефона)
- Выбор доступной даты и времени для записи
- Подтверждение записи
- Автоматическое напоминание за день до занятия

### Для администратора:
- Управление слотами (добавление, блокировка)
- Просмотр всех записей
- Просмотр данных клиентов
- Аналитика по записям

## Установка и запуск

1. Клонируйте репозиторий: git clone /AlexGIT1999/telegram_bot_zanyatiya

2. Создайте виртуальное окружение:python -m venv venv

3. Активируйте виртуальное окружение:
- Windows: `venv\Scripts\activate`
- macOS/Linux: `source venv/bin/activate`

4. Установите зависимости: pip install -r requirements.txt

5. Создайте файл `.env` и добавьте туда:
BOT_TOKEN=ваш_токен_от_BotFather 
ADMIN_ID=ваш_telegram_id

6. Запустите бота: python bot.py

## Структура проекта

- `bot.py` - основной файл бота
- `data.py` - работа с данными
- `handlers/` - обработчики сообщений
- `client_handlers.py` - обработчики для клиентов
- `admin_handlers.py` - обработчики для администратора
- `.env` - конфигурационный файл (не добавляется в репозиторий)
- `users.json` - данные пользователей
- `bookings.json` - записи на занятия
- `slots.json` - доступные слоты для записи

## Разработка

Проект написан на Python с использованием библиотеки pyTelegramBotAPI.
